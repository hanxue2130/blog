}

Q2 How many prime numbers there are between 101-200, and output all prime numbers.

public static void primeNum() {
    int count = 0;
    for (int i = 101; i < 200; i++) {
        if(priHelper(i)){
            count +=1;
            System.out.println(i);
        }
    }
    System.out.println("The total prime number between 100 to 200 is: "+count);
}
public static boolean priHelper(int n){
    if(n<=1){
        return false;
    }
    //forgot sqrt && equal
    for (int i = 2; i <= Math.sqrt(n); i++) {
        if(n%i==0){
            return false;
        }
    }
    return true;
}

Q3 Print out all the “daffodil numbers”, the so-called “daffodil numbers” refers to a three-digit number whose cubic sum of the digits is equal to the number itself. For example: 153 is a “daffodil number”, because 153=1 cube + 5 cube + 3 cube.

public static void NarcissisticNumber(){
    for (int i = 100; i < 999; i++) {
        int unit = i % 10;
        int decade =  i /10 % 10;
        int hundred = i / 10 / 10;
        if(i == unit * unit * unit + decade* decade * decade + hundred * hundred * hundred){
            System.out.println(i);
        }
    }
}

Q4: Decompose a positive integer into prime factors. For example: input 90, print out 90=233*5.

public static void DecompositionPrimeFactor() {
    int n = 90;
    for (int i = 2; i < Math.sqrt(n); i++) {
        if(n%i ==0){
            System.out.print(i+"*");
            n/=i;
            i--;
        }
    }
    System.out.println(n);
}
